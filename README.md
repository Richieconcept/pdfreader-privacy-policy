# PDFReader Privacy Policy

This folder is a static privacy-policy site for Google Play.

## Deploy on Vercel

1. Push this project to GitHub.
2. In Vercel, create a new project from the GitHub repo.
3. Set the Vercel root directory to `privacy-policy`.
4. Leave the framework preset as `Other`.
5. Leave the build command empty.
6. Leave the output directory empty.
7. Deploy.

After Vercel gives you the live URL, paste that URL into Play Console:

`Policy > App content > Privacy policy`

The URL must stay public, active, non-gated, and must not point to a PDF file.
